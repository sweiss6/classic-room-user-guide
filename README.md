# Classic Room AV System User Guide

Public download site for the Waukesha County Technical College Classic Room AV
System User Guide.

The QR code should point to the stable GitHub Pages address:

`https://sweiss6.github.io/classic-room-user-guide/`

To publish an updated guide, replace
`Classic-Room-AV-System-User-Guide.pdf` without changing its filename, update
the version and date in `index.html`, then commit and push to `main`.
